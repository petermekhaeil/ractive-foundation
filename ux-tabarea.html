ux-tabarea

Tabs are elements that help you organize and navigate multiple documents in a single container. They can be used for switching between items in the container.

Read Foundation tabs.html page for details.

Markup example:



    
        Tab One
        Tab Two
        Tab Three
    

    
        
            First tab content
        
        
            Second tab content
        
        
            Third tab content
